# Movies_Management


HOW TO RUN:

# run nodeJS dev server (in different shell)
> cd Server
> npm i
> node index

# run React-vite dev server (in different shell)
> cd Client/myapp/
> npm i
> npm run dev

open react app in web:
http://localhost:5173/
